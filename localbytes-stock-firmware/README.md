# LocalBytes Stock Tasmota Firmware

This firmware file is from LocalBytes and contains the firmware that they ship with the bulb.

This is what comes on the bulb when it comes out the box. It is Tasmota 11.1.0 with the official template integrated into it.

Flashing this firmware to the bulb will return it to how it came out the box.

Thank you to LocalBytes for providing this file.
